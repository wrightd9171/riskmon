Scheduling (external Windows task)

The recurring send is driven by Windows Task Scheduler, not this app, so it runs even when the app is closed. Register it once from the project root:

powershell -ExecutionPolicy Bypass -File scripts\register-notify-task.ps1

It prompts for your master password and the day/time, then creates a task that runs send_digest.py on that cadence. Because the task must decrypt these keys without you present, it has to hold your master password — see the README for the exact mechanism and the security tradeoff.

Fidelity

Fidelity has no official API. This app pulls positions two ways, in order:

  1. Automated login (optional) via the fidelity-api browser scraper — configure it below.
  2. CSV fallback — export from fidelity.com and drop the file in data/.

Each refresh tries the login first and falls back to the CSV if it can't complete (library not installed, 2FA it can't satisfy, etc.). The API gives live quantities and prices; cost basis is merged in from your latest CSV (average cost/share × current shares), so keep a recent CSV in data/ for accurate Fidelity P&L.

Automated login (fidelity-api)

One-time install on this machine: pip install fidelity-api then playwright install firefox. Credentials are encrypted with your master password. The optional TOTP secret lets it clear authenticator-app 2FA unattended — required for the scheduled/daily runs, since those can't prompt you for a code.

Fidelity login

Using Duo push, or no TOTP secret? Leave the TOTP box blank, save your username/password above, then run the one-time interactive login from the project root. It opens a browser so you approve 2FA once, then saves the session so later (headless) refreshes skip 2FA:

.venv\Scripts\python.exe fidelity_login.py

Re-run it whenever refreshes start falling back to CSV — that means the saved Fidelity session expired.

CSV fallback

Export your positions from fidelity.com (Positions → Download) and drop the CSV into data/ — any filename starting with Fidelity_. Refresh uses it whenever the automated login isn't available.
